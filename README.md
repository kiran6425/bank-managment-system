# 🏦 Bank Management System

A modern and feature-rich desktop application developed using **Java Swing, AWT, JDBC, and MySQL** to simulate real-world banking operations through an interactive ATM-style interface.

This project is designed to provide a secure, efficient, and user-friendly banking environment where users can perform essential banking activities such as account creation, deposit, withdrawal, balance enquiry, mini statement generation, PIN change, and fast cash transactions.

Built with a strong focus on performance, maintainability, and user experience, the application demonstrates practical implementation of **Java GUI development, database connectivity, object-oriented programming, and secure transaction handling techniques**.
